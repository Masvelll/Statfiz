#### Соотношение Стокса-Эйнштейна

Данное соотношение имеет следующий вид:

$$D = \dfrac{kT}{6 \pi \eta R}$$

Поэтому в таком виде


$$\eta = \dfrac{kT}{6\pi R} \dfrac{1}{D}$$

Будем иметь линейную зависимость

![](./images/R.png)
